# iPadOS Calculator

Meet your new favorite calculator for iPad! 🎉  
Designed with a clean iPad-style interface, this app handles everything from simple calculations to advanced scientific functions.

**Features:**
- Full iPad layout — right column neatly holds Backspace, ÷, ×, −, +
- Bottom row: 0, ., = — quick and intuitive
- Toggle between basic and scientific modes easily
- Dark theme with vibrant iOS-style orange buttons
- Backspace and full calculation history at your fingertips
- Universal support for iPhone & iPad (`UIDeviceFamily = 1,2`)

# iPadOS 计算器

专为 iPadOS 设计的功能齐全计算器。界面简洁，操作流畅，无论基础计算还是科学计算都能轻松完成。

**功能特点:**
- iPad 全屏布局 —— 右侧竖排按钮：退格、÷、×、−、+，清晰整齐
- 底部一行按钮：0、.、=，使用直观又方便
- 可切换基础/科学计算模式
- 深色主题，iOS 风格橙色按钮
- 支持退格操作和完整历史记录
- 通用设备支持，iPhone/iPad 都可用 (`UIDeviceFamily = 1,2`)
